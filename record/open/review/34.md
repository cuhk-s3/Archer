- **Expected Issue**: The compiler might crash or generate incorrect code because it attempts to hoist a predicated load without properly handling the predication mask.

- **Expected Issue**: The compiler might crash or generate incorrect code because it attempts to hoist a load that is not a single scalar, or it might fail to hoist a valid single-scalar load.
