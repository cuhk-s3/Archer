### Strategy 1: fdiv_self_denormal_flush_to_zero

- **Target**: A test case with `fdiv float %x, %x` where `%x` is known to be non-zero but could be a denormal, and the function has a denormal-fp-math attribute that flushes denormals to zero.
- **Rationale**: The `fdiv_self` function assumes `X / X` is exactly 1.0 or NaN. It checks `KnownSrc.isKnownNeverLogicalZero(Mode)` to rule out NaN. If the input is a denormal and the mode flushes denormals to zero, `isKnownNeverLogicalZero` should return false.
