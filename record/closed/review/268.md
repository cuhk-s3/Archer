- **Target**: A test case with a matrix intrinsic that lowers to a select with a vector condition and profile metadata.
- **Rationale**: The fix propagates profile metadata for selects with scalar conditions but explicitly asserts that vector conditions should not have profile metadata propagated. - **Expected Issue**: The select instruction will be created without the profile metadata, leading to incorrect profile information propagation.

### Strategy 2: Test Select with Scalar Condition and Profile Metadata

- **Target**: A test case with a matrix intrinsic that lowers to a select with a scalar condition and profile metadata.
- **Rationale**: The fix propagates profile metadata for selects with scalar conditions. We need to ensure that the metadata is correctly propagated and that it doesn't cause any issues in the lowered code.
- **Expected Issue**: The select instruction will be created with the profile metadata, which might be incorrect or unexpected for scalar conditions in some contexts.
