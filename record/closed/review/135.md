## Test Strategies

### Strategy 1: Test fdiv with unknown profile

- **Target**: `llvm/test/Transforms/InstCombine/simplify-demanded-fpclass-fdiv.ll`
- **Rationale**: The patch changes `Builder.CreateSelectFMF` to `Builder.CreateSelectFMFWithUnknownProfile` in `InstCombinerImpl::SimplifyDemandedUseFPClass` for `Instruction::FDiv`. This means that when a select is created to handle the case where the source is 0, inf, or nan, it should now have an explicitly unknown profile. We need to test that this metadata is correctly attached to the generated select instruction.
- **Expected Issue**: The generated select instruction will not have the explicitly unknown branch weights metadata, which might cause issues in downstream passes that rely on this metadata for profiling.
